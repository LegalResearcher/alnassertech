## 🛠 التقنيات المستخدمة (Tech Stack)

تم بناء مشاريعنا باستخدام "مكدس تقني" حديث يضمن الأداء العالي وتجربة مستخدم لا تضاهى:

- **Vite**: لسرعة بناء وتشغيل خيالية.
- **TypeScript**: لضمان كود برمجي قوي، آمن، وقابل للتطوير.
- **React**: لبناء واجهات مستخدم تفاعلية وسلسة (SPA).
- **shadcn-ui**: مكتبة مكونات تصميم احترافية وأنيقة.
- **Tailwind CSS**: لتصميم متجاوب يعمل بكفاءة على جميع الأجهزة (Responsive Design).

## 💻 دليل التشغيل للمطورين (Setup Guide)

إذا كنت مطوراً وترغب في تشغيل المشروع محلياً، اتبع الخطوات التالية:

**المتطلبات:** تأكد من تثبيت [Node.js](https://nodejs.org/) و [Git](https://git-scm.com/).

```bash
# 1. استنساخ المستودع (Clone)
git clone [https://github.com/LegalResearcher/alnassertech.git](https://github.com/LegalResearcher/alnassertech.git)

# 2. الدخول إلى مجلد المشروع
cd alnassertech

# 3. تثبيت الحزم والاعتماديات
npm i

# 4. تشغيل السيرفر المحلي (للمعاينة)
npm run dev
